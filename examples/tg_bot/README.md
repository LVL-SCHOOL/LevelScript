# Telegram Bot на языке LevelScript
## Простой Telegram-бот, написанный на языке программирования LevelScript.

## Описание

### Бот умеет:
- Отвечать на приветствия (привет, здравствуйте и т.д.)
- Отвечать на вопрос "как дела?"
- Работать как эхо-бот для непонятых сообщений
- Обрабатывать команду /start

### Структура проекта
```text
📁 examples/
└── 📁 tg_bot/ - Пример Telegram-бота на языке Law
    ├── 📁 bot/ - Основные компоненты бота
    │   ├── 📄 sender.raw - Модуль отправки сообщений через Telegram API
    │   └── 📄 updater.raw - Модуль получения обновлений (long polling)
    ├── 📁 config/ - Конфигурация и настройки
    │   ├── 📄 parser.raw - Парсер .env файла с настройками
    │   └── 📄 settings.raw - Класс настроек бота
    ├── 📄 .env - Файл с реальными настройками (токен, адрес API)
    ├── 📄 .env_sample - Пример файла настроек
    ├── 📄 handlers.raw - Обработчики текстовых сообщений
    ├── 📄 main.raw - Главный файл, точка входа в приложение
    ├── 📄 README.md - Документация проекта
    └── 📄 tests.raw - Тесты для проверки функционала
```

###  Настройка
1. Создайте файл .env в корне проекта:

```
ТОКЕН=ВАШ_ТОКЕН_ТЕЛЕГРАМ_БОТА
АДРЕС=https://api.telegram.org/bot
```

2. Получите токен бота у @BotFather

## Запуск
```
law.exe --run examples\tg_bot\main.raw
```

## Запуск тестов
```
law.exe --run examples\tg_bot\tests.raw
```

### Особенности реализации
- Использует long polling для получения обновлений
- Обработка сообщений происходит в фоновом режиме
- Автоматические повторные попытки при ошибках сети
- Очередь сообщений для обработки

### Зависимости
Проект использует стандартные библиотеки LevelScript
